# Email Sender - .NET

Aplicação desktop em **C# WinForms** (.NET 4.7.2) para envio de e-mails com suporte a múltiplos destinatários e anexos.

## Funcionalidades

- Envio para múltiplos destinatários (separados por `;`)
- Anexo de múltiplos arquivos
- Pré-visualização de anexos (imagens)
- Exclusão individual de anexos
- SMTP Gmail com SSL

## Stack

- C# / .NET Framework 4.7.2
- WinForms
- System.Net.Mail (SMTP)

## Como executar

Abra `EnviarEmail.sln` no Visual Studio e execute (F5).

> **Nota:** Configure suas credenciais SMTP antes de usar.
